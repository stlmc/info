  • Benchmark models 

    There are three kinds of continuous dynamics: linear dynamics, polynomial dynamics, and ordinary differential equations (ODEs). You can download the model file using the below link.

    • Car: Linear [link], Polynomial [link], Ode [link]
    • Thermostat: Linear [link], Polynomial [link], Ode [link]
    • Watertank: Linear [link], Polynomial [link], Ode [link]
    • Railroad: Linear [link], Polynomial [link]
    • Airplain: Ode [link]
    • Battery: Linear [link], Polynomial [link], Ode [link]
    You can see dynamics and STL formulas for each model in the attached file. [models.pdf]

  • Experimental Evaluation 

    For experimentally evaluation, we compare the performance of our algorithm with the previous algorithm using our tool stlmc STL Bounded Model Checker. We use above models for bounded model checking (Experiment 1) and STL formulas for STL satisfiability checking (Experiment 2). You can download STL formulas of Experiment 2 using the following links. [nested-1.model, nested-2.model, nested-3.model]
