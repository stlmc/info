STLMC : STL Bounded Model Checker

    stlMC is a bounded model checking tool for signal temporal logic (STL) of hybrid system. The algorithm of the tool is refutationally complete for STL properties of bounded signal. It encodes STL bounded model checking problems as the reductions into the satisfiability of a first-order logic formula over the real numbers. The satisfiability of the resulting formula can be determined using existing SMT solvers such as Z3, Yices, and dReal3.

    Installation 

    STLMC is provided as a python package. To install the tool, run the following instruction:

    pip install -i https://test.pypi.org/simple/ stlmc==0.1.0.dev4

    The source code can be downloaded through the attached file [source.zip].

    Usage 

    To perform model checking of STL on the above model from bound 1 to 5 using dReal3, run the following command:

    ./stlmc -lower 1 -upper 5 -solver dreal thermostat.model
